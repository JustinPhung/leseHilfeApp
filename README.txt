Folgende Sachen musst du installieren um die App zum laufen zu bringen.

Node.js
Ionic 3 
Angular 4 
Cordova

Dann gehst du im Terminal in diesen Ordner rein und führst folgenden Befehl aus

ionic serve -l

Es geht dann automatisch ein Fenster auf welches Android und Ios simuliert. 
Damit kannst du dir die App anschauen.

Um es auf deinen Android laufen zu lassen:

ionic cordova run android --prod

Um es auf einem Ios System laufen zu lassen:

ionic cordova run ios --prod
